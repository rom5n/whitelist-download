# 🌊 whitelist-download

Этот скрипт предназначен для автоматического скачивания бесплатных VLESS-конфигов (из открытых GitHub-репозиториев) и создания из них единой ссылки-подписки (Subscription link) для вашего VPN-клиента. Скрипт позволяет обходить блокировки, собирая и предоставляя актуальные конфигурации в удобном виде.

## 🌊 3800+ Конфигов
## ⚠️ Ссылка на подписку не работает в мобильной версии v2RayTun
## ⚠️ Иногда антивирус жалуется при первом запуске

## ✨ Основные возможности

- **Автоматическое обновление:** Скрипт скачивает свежие конфигурации каждый час.
- **Локальный сервер подписки:** Создает локальный HTTP-сервер для раздачи конфигов, который легко подключить к любому современному клиенту (v2rayN, Nekobox, Hiddify, v2rayNG и др.).
- **Автозагрузка (Windows):** Скрипт автоматически прописывается в реестр и незаметно стартует вместе с системой.
- **Управление количеством конфигов:** Поддержка лимитов и смещений для подписки (чтобы ваш VPN-клиент не завис от гигантского количества подписок).

## 🚀 Использование

### 1. Установка в Windows
1. Скачайте архив wl-download.zip и разархивируйте оттуда .exe файл.
2. Переместите exe-шник любую удобную папку
3. При первом запуске программа автоматически:
   - Добавит себя в автозагрузку Windows
   - Скачает актуальные конфиги из сети
   - Создаст рядом с собой в папке файлы: log.txt и configs.txt
   - Запустит локальный сервер подписки на порту `55000`

### 2. Подключение подписки в VPN-клиент
Чтобы использовать скачанные сервера в вашем приложении (например, **v2rayN**, **Nekobox** или **Hiddify**):
1. Откройте файл `log.txt`, который появится в папке с программой после запуска.
2. Найдите в логах строку, похожую на эту: 
   `✨✨ Check it: http://192.168.Х.ХХ:55000/sub` (IP-адрес будет вашим локальным).
3. Скопируйте эту ссылку, она постоянная.
4. Если вы добавляете подписку на телефоне, то убедитесь, что вы подключены к тому же WiFi, что и ПК, а также у вас выключены все VPN/Proxy
5. Откройте ваш VPN-клиент на ПК или телефоне и вставьте ранее скопироваую ссылку для добавляния подписки или импортируйте ее из буфера обмена
6. ❗ Если выдает ошибку, то возможно программа показала неверный IP адрес вашего ПК. Зайдите в настройки WiFi и снизу напротив строки 'IPv4-адрес' будет нужный IP, замените его в ссылке
7. Готово! В вашем клиенте появятся конфиги серверов для обхода белых списков

## ⚙️ Фишки

Если серверов скачивается слишком много и вашему клиенту тяжело их обрабатывать, вы можете ограничить их количество, просто дописав цифры в конец ссылки-подписки:

- `http://IP:55000/sub/50` — получить только первые **50** конфигов.
- `http://IP:55000/sub/10-30` — начиная с **10-го** получить **30** конфигов (первые 9 будут пропущены).
- `http://IP:55000/sub` — ❗ Будут импортированы **все** подписки ❗

## 📂 Файлы
- `configs.txt` — база сохранённых конфигураций в формате `vless://`. Скрипт полностью перезаписывает этот файл каждый час свежими данными.
- `log.txt` — логи работы программы. Сюда пишутся возможные ошибки, адреса серверов и информация об обновлениях. 

> ⚠️ **Внимание:** Не перемещайте `.exe` файл после первого запуска, иначе автозагрузка Windows перестанет его находить при старте системы. Если вам необходимо перенести программу в другую папку — просто запустите её один раз на новом месте, и путь в реестре обновится автоматически.
